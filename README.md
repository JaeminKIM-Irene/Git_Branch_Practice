# counterfeit_trademark

## 환경 관리
### 01. 스크립트 설명
- __init_dev_env.sh__
  - 코드 개발 및 실행을 위한 패키지 설치 스크립트
  - 파이썬 환경에서 <code>counterfeit_trademark/</code>에 구성 된 코드 로드 가능 ex\)
    ```python
    import counterfeit_trademark
    ```
    

### 02. 환경 설정 가이드
- __최초 설정__
  ```
  <!-- init for Mac or Linux -->
  $ source init_dev_env.sh

  <!-- init for Window -->
  $ init_dev_env.bat
  ```
